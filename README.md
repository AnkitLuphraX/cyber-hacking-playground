## 🖧 About the Portal

**CYBER_NET // NODE_TERMINAL v2.0** is a state-of-the-art interactive retro-cyber hacking terminal simulator. Designed with scanline filters, screen flicker overlays, glassmorphic HUD panels, and interactive Canvas widgets.

All audio click transients, ambient space cockpit drone sweeps, and success chord arpeggios are **synthesized procedurally on-the-fly using the native Web Audio API**—meaning zero network asset-loading delay and 100% offline capability.

## 📦 Setup & Installation Instructions

Follow these simple steps to run the interactive console locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/AnkitLuphraX/cyber-hacking-playground.git
   cd cyber-hacking-playground
   ```

2. **Install project dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **Verify bundle compilation**:
   ```bash
   npm run build
   ```

Open **`http://localhost:5173/`** in your browser to launch the portal.
